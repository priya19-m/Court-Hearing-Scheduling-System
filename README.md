## Problem Statement
The objective of this project is to design and implement a Court Hearing Scheduling System using Java with an object-oriented approach.

The system allows:
1.Managing courts, judges, lawyers, and staff.
2.Creating and scheduling hearings with date, time, and courtroom allocation.
3.Tracking hearing status (Scheduled, Adjourned, Completed, Canceled).
4.Recording case details and attaching relevant documents.
5.Sending notifications to judges, lawyers, and staff for upcoming hearings.
6.Generating reports on hearings, case statuses, and workloads.
7.Handling invalid data and missing entities through exception handling.
## How to Run in Eclipse IDE:
Step 1: Open Eclipse IDE
Download from https://www.eclipse.org/downloads/.
Step 2: Create a New Project
Go to File → New → Java Project.
Enter project name: CourtHearingScheduler.
Click Finish.
Step 3: Create Packages
Right-click on src → New → Package. Create these packages:
com.court.model, com.court.service, com.court.exception, com.court.main
Step 4: Add Classes
In each package, create the corresponding .java files. Copy-paste the full code from the repository into these files.
Step 5: Run Main Class
Go to com.court.main → CourtSchedulerMain.java.
Right-click → Run As → Java Application.
